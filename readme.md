# Conteúdo do repositório

Esse repositório irá conter as aulas sobre linguagem C que foram ministradas no curso de Engenharia de Computação, na Universidade Federal do Rio Grande do Sul (**UFRGS**).
Aqui estarão resolvidos os exercícios propostos durante as aulas e os respectivos problemas semanais, que eram desenvolvidos em laboratório a fim de validar os conhecimentos adquiridos.
